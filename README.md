# 🔍 Online Lost Item Tracking System (OLITS)

A premium, full-stack web application designed for campus environments (like MMU Mullana) to help users report and track lost or found items efficiently.

## ✨ Features
- **Modern UI/UX:** High-tech, responsive design with dark mode support.
- **Real-time Match Alerts:** Automatically notifies users if a matching item is found.
- **Admin Dashboard:** Centralized console for managing reports and viewing campus "Heat Maps".
- **Secure Access:** Verification via 4-digit PIN for updating or deleting reports.
- **Gibberish Detection:** Smart filtering to prevent spam submissions.

## 🚀 Tech Stack
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Backend:** Node.js, Express.js
- **Database:** MariaDB (Relational Database)
- **Deployment:** Render (Live Server)

## 🛠️ Local Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Setup your MariaDB database and configure the credentials in the `.env` file.
4. Run `npm start` to launch the server.
5. Open `http://localhost:5000` in your browser.
